# E-Commerce Sales Data Analysis Using SQL

## Description
This project analyzes an e-commerce sales dataset using SQL queries. The analysis focuses on sales performance, customer behavior, product trends, revenue generation, and order status insights. SQL was used to extract meaningful business insights from the dataset and answer key analytical questions.

## Tools Used
- MySQL Workbench

## Dataset
The dataset contains information about:
- Orders
- Customers
- Products
- Sales Revenue
- Quantities Sold
- Order Status

## Key Analysis Performed
1)Retrieved and explored data using SELECT queries.
2)Filtered records with WHERE conditions.
3)Sorted data using ORDER BY.
4)Grouped data by categories using GROUP BY.
5)Calculated totals using COUNT() and SUM().
6)Determined averages using AVG().
7)Identified top-performing products and sales trends.
8)Generated insights to evaluate overall business performance.

## How to Run the Project
1. Open MySQL Workbench.
2. Create a new database.
3. Import the dataset into MySQL.
4. Open the SQL script file:' SQL Decodelab project3.sql.'
5. Execute the queries.
6. Review the results generated by each query.
